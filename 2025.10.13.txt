- kali (Docker 설치)
$ apt update && apt upgrade
$ apt install docker.io
// curl 은 Text 기반의 웹 접속 가능, 및 여러 RESTful API 전송 가능하게 해주는 프로그램
$ sudo dnf install -y curl 
// 원격서버에서 파일을 다운로드 받는다.  
$ sudo curl -L "https://github.com/docker/compose/releases/download/v2.29.2/docker-compose-linux-x86_64" -o /usr/local/bin/docker-compose
$ sudo chmod +x /usr/local/bin/docker-compose
$ docker-compose version


- Ubuntu (랜카드 추가, LAN segment)
1) Desktop 설치
apt update && apt upgrade

2) tilix 설치
apt install tilix -y
3) VM tools 설치
